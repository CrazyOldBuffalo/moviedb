
//  Application   //
Java application covers a movie system that contains a user, movies and ratings collection.
The System links the movies and users file's to ratings displaying a rating a user gives on a film.
System displays the average rating for each film in the movies collection by users.


//  Data Sets    //
users.dat contains all records of users within the Movie system.
ratings.dat contains all ratings of movies from movies.dat given by each user in user.dat
movies.dat contains all records of the different movies within the Movie System
